# rovas — Roadmap

## v0.1.0 — MVP (current)

- [x] Full Latin → Rovás transliteration
- [x] Rovás → Latin reverse (best-effort)
- [x] All 14 Hungarian vowels incl. long vowels
- [x] All digraphs: cs, sz, zs, gy, ny, ly, ty, dz, dzs
- [x] Vowel omission option (classical mode)
- [x] Long vowel option (modern vs classical)
- [x] Number conversion (quasi-Roman additive system)
- [x] `isRovas` / `isLatin` detection utils
- [x] `toCodePoints` debug util
- [x] Dual ESM + CJS build (tsup)
- [x] TypeScript types + declaration files
- [x] Vitest test suite with real Hungarian words & place names
- [x] Hungarian README
- [x] Examples folder

## v0.3.0 — Correctness & Edge Cases

- [ ] Positional K variants: `EK` (word-initial/medial) vs `AK` (word-final) — U+10CD3 vs U+10CD4
- [ ] `MP` ligature (EM+EP) — U+10CC3 — replaces standalone `m` before `p`
- [ ] `NT` / `ND` variant letters — U+10CC5 (ENC), U+10CC8 (AND)
- [ ] Punctuation: mirrored `?` and `,` in RTL contexts
- [ ] Handle `Q`, `W`, `X`, `Y` (foreign letter ligature compositions per oldhungarian.eu)
- [ ] Case-preserving token metadata (track original casing for display)
- [ ] Strict mode: throw on unrecognised characters instead of passthrough
